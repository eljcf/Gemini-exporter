# Privacy Policy for Gemini Chat Exporter

**Last Updated:** February 2026

## 1. Data Collection
This extension (**Gemini Chat Exporter**) does **not** collect, store, share, or transmit any user's personal information, browsing history, or conversation data.

## 2. Data Usage
All data processing (parsing chat history, formatting Markdown, identifying files) happens entirely **locally** within your web browser (Client-Side). No data is sent to any external servers or third-party analytics services.

## 3. Permissions
The extension requests the following permissions solely for its core functionality:
* **activeTab / Host Permissions**: To access the text content of the Google Gemini web page (`gemini.google.com`) only when you explicitly trigger the export action.
* **scripting**: To execute the auto-scroll and extraction scripts required to retrieve your chat history.

## 4. Contact
This is an open-source project. If you have any questions, please verify the source code at our GitHub repository.
